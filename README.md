### Алгоритм решения



###### 



#### Алгоритм работы программы начинается с запуска метода Main, где пользователю предлагается последовательно ввести три целых числа для создания базового объекта. После считывания данных через консоль программа инициализирует экземпляр класса MyNumbers, передавая введенные значения в конструктор для заполнения внутренних полей. Далее вызывается метод вычисления максимальной из последних цифр этих чисел, который работает путем взятия остатка от деления каждого числа на десять и последующего их сравнения. Результаты работы и содержимое полей выводятся на экран благодаря перегруженному методу ToString. Затем программа демонстрирует работу конструктора копирования, создавая новый независимый объект на основе уже существующего и отображая его данные. На следующем этапе пользователю предлагается ввести параметры для дочернего класса Box, представляющего коробку. Программа создает объект этого класса, где три числа интерпретируются как длина, ширина и высота. С помощью встроенных методов рассчитывается объем коробки как произведение сторон и выполняется логическая проверка на принадлежность объекта к кубу через сравнение всех измерений. В завершение на консоль выводится полная информация о коробке через перегруженный метод ToString, который объединяет текстовое описание базовых полей и новые расчетные данные, после чего выполнение программы приостанавливается до нажатия клавиши пользователем.

### Задача 1



### 



### Текст задачи







#### LineSegment - Унарные операции:

#### ! - вычислить длину отрезка, результат должен быть типа double;

#### ++ увеличить координаты границ отрезка на 1.

#### Операции приведения типа:

#### int (явная) – результатом является целая часть координаты х;

#### double (неявная) – результатом является координата y.

#### Бинарные операции:

#### +int d, целое число (лево- и правосторонняя операция, координаты границ

#### увеличиваются на число d);

#### < LineSegment, целое число – результат равен true, если целое число

#### попадает в заданный отрезок и false–в противном случае.







### Алгоритм решения





#### В методе Main программа создаёт объект класса Zadan3 на основе введённых данных и выполняет серию тестов перегруженных операций. Сначала через унарный оператор вычисляется длина отрезка как абсолютная разность его границ, после чего инкремент увеличивает обе координаты на единицу. Далее программа демонстрирует преобразование объекта в стандартные типы, извлекая целую часть первой координаты и значение второй. Затем выполняется бинарное сложение для смещения всего отрезка на целое число и вызывается оператор сравнения, который проверяет вхождение числа в текущий диапазон. Весь процесс сопровождается выводом промежуточных состояний объекта в консоль для наглядности работы операторов.
